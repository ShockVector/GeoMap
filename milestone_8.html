Maps of Albuquerque

Terrain Bing Aerial Map with Labels Paper Map

These maps of Albuquerque are a test in openlayers capabilities. I chose the Stamen terrain map as the base map because it is a good mix of aesthetic appeal and vector information. The Bing aerial image map serves as a raster comparison map. The "paper" map is a repurposed Stamen toner map with no background. It so happened to overlay with the background of the webpage so I kept it for novelty.
